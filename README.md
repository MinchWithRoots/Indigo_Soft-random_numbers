# Генератор Excel-файла со случайными числами

Простой PHP-скрипт, который создаёт Excel-файл **`random_numbers.xlsx`** с таблицей **10×10**, заполненной случайными числами от **1 до 100**, и применяет базовое форматирование:
- Тонкая обводка вокруг всех ячеек  
- Выравнивание текста по центру (по горизонтали и вертикали)  
- Автоматическая ширина колонок для корректного отображения  

---

## Требования

- **PHP 8.1 или новее**
- Необходимые PHP-расширения: `gd`, `zip`, `fileinfo`
- [Composer](https://getcomposer.org/) (для управления зависимостями)

---

##  Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/MinchWithRoots/Indigo_Soft-random_numbers.git

2. **Установите зависимости**
   ```bash
   composer install
   
3. **Запустите скрипт**
      ```bash
      php index.php

## После выполнения:

    Файл random_numbers.xlsx появится в папке excel_files/
    В консоли вы увидите:
    файл успешно создан: /путь/к/proj/excel_files/random_numbers.xlsx

    ⚠️ Если файл уже существует, скрипт завершится с сообщением:
    файл уже существует: ... — чтобы избежать случайной перезаписи.

## Структура проекта
- index.php             - основной скрипт
